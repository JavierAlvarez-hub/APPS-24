jerarquia de Css. - restricciones

los navegadores aplican estilos a los documentos HTML por jerarquia basada en :

  1. La cascada
  2. La especificidad
  3. La herencia

La cascada es el procedimiento con el que un navegador aplica los estilos y el orden de importancia

  1. Origen del codigo
    • Reglas del user Agent
    • Reglas del usuario
    • Reglas del desarrollador
  2. La especificidad del selector
  3. Orden de aparacion

la especificidad es la fuerza restrictiva o peso de un selector

  • Estilo en linea1
  • Identificadores 0,1
  • Clases, atributos, pseudoclases 0,0,1
  • Etiquetas HTML 0,0,0,1
  • !important. Salta toda la jerarquia

La herencia es la capacidad que tienen los elementos padre de hacer heredar a sus hijos sus propiedades. valores inherit o heredados
