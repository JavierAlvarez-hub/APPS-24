Etiquetas semanticas estructurales

div:Es una etiqueta contenedora que no tiene semántica
header:Cabecera de un sitio web o de una sección
main:Define la sección principal del documento, sólo puede existir una etiqueta main por documento
footer:Pié de página de un sitio web o de una sección
nav:Representa una navegación
article:Representa contenido dependiente de una seccion
aside:Representa contenido complementario o secundario
section:Representa una sección de contenido genérico
address:Representa una información de contacto

Etiqueta de bloque VS Etiqueta de linea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de línea con sus etiquetas hermanas.

La etiqueta de línea por excelencia es la span, una etiqueta de línea sólo ocupa el espacio necesario que tiene su contenido. Hola soy otra span hola soy otra span

Imagenes

en HTML podemos utilizar imagenes sin perdida de calidad en la compresion de su composicioncomo: GIF,png o con perdida de calidad con JPG

GIFGIF; Formato sin perdedida de calidad con capacidad para comprimir hasta 256 colores, valido para tintas planas, logotipos, etc... Acepta animaciones.

PNG-8: igual a GIF y su ratio de compresion es mejor que el de un archivo GIF. No acepta animaciones.

JPG: formato perfecto para cuatricomas e imagen fotografica, pero tiene perdida de calidad

PNG-24: Formato perfecto para fotografias y transparencias a color real, tiene muy poca perdida de calidad.

SVG: formato de imagen vectorial , sin perdida y la mas recomendable para logotipos e iconos
